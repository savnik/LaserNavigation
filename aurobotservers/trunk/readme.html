Introduction

To install see the install.html. install.html

This is a set of server applications autonomous robot control. The main servers are a camera server (ucamserver) a laser scanner server (ulmsserver) and an empty server (userver). The servers has a set of modules that can be loaded (module load=XXX) and the possibility to load plug-in module compiled separately. There is a monitoring client application (uclient and auclient) to monitor the data from the servers graphically.

The functions are avilable to the robot using a socket connection. The protocol is text-based in a XML format.

A test session could be like this: Start a server (from the bin directory): 

[jca@pc89 sample]$ ./ucamserver
      --- pc89 is open on port 24920 for requests.
      >>
The server is now accepting connections on port 24920.

Start a telnet client connection: 

[jca@pc89 jca]$ telnet localhost 24920
Trying 127.0.0.1...
Connected to localhost (127.0.0.1).
Escape character is '^]'.

Type in e.g.: 

 <camGet name size/>

Reply is something like: 

<camget posName="Left" height="240" width="320" name="Philips 740 webcam"/>
